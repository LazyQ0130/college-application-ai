# 雪峰志愿分析助手

基于录取数据与公开志愿填报方法论的高考志愿分析网页工具。

它不是只会聊天的普通 AI。项目内置 24 省官方录取数据库，可根据省份、分数、位次、选科和专业偏好查询历史数据，并从专业选择、冲稳保分层和填报风险等角度给出初步分析。

## 核心能力

- 查询 24 省历史录取数据
- 分析专业就业、学习难度、家庭资源和升学路径
- 按“冲、稳、保”解释志愿分层逻辑
- 支持 OpenAI 兼容 API，可配置 DeepSeek 等模型
- 可选配置 Tavily，联网补充最新招生和行业信息
- 多轮聊天记录保存在浏览器本地
- 支持深色模式和移动端布局

## 本地运行

需要 Python 3.10 或更高版本，无需安装第三方依赖。

Windows 用户可以直接双击：

```text
启动.bat
```

也可以在项目目录执行：

```powershell
py -3 server.py
```

启动后访问：

```text
http://127.0.0.1:8766/
```

首次运行会将 `admission_clean.db.gz` 自动解压为本地数据库。生成的 `.db` 文件已被 Git 忽略。

## API 配置

打开网页右上角的“API设置”，填写：

- Base URL：例如 `https://api.deepseek.com`
- API Key：对应模型服务的密钥
- Model：例如 `deepseek-chat`
- Tavily Key：可选，用于联网检索

密钥只保存在当前浏览器的本地存储中，不会写入仓库。

## 项目文件

```text
server.py                 单文件 Web 服务、页面、聊天与数据查询逻辑

img_suit.png              页面头像素材
启动.bat                   Windows 启动脚本
打开我.html                服务状态检测与跳转页
```
